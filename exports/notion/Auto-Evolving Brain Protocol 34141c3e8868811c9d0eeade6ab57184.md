# Auto-Evolving Brain Protocol

## Objective

Turn captured information into durable understanding and reusable references.

## Loop (Daily + Weekly)

1. Capture raw signals into `Inbox`.
2. Convert useful signals into `Knowledge` notes.
3. Distill one claim or principle into `Memory Graph`.
4. Link each node to at least one source and one existing note.
5. Review and strengthen weak nodes weekly.

## Reference Rules

- Every enduring note gets a stable `Canonical ID`.
- Every claim needs evidence (`Source URL` or linked note).
- Upgrade `Understanding` over time: Surface -> Working -> Deep -> Teach.
- Keep notes atomic: one idea per note whenever possible.

## Evolution Triggers

- If the same insight appears 3+ times, create a canonical pattern node.
- If a task repeats 3+ times, create a reusable how-to note.
- If confidence is low, schedule a review and add missing sources.

## Weekly Upgrade Checklist

- Promote at least 2 notes to higher understanding level.
- Merge duplicates.
- Archive stale or low-value notes.
- Add one new cross-link between distant topics.
